Key Features

Tabular Optimization

Schema declared once, rows follow without field repetition. Perfect for datasets.

Smart Compression

Numbers abbreviated (95K), booleans compact (T/F), nulls minimized (N).

LLM Optimized

Designed for AI tokenization patterns. Reduce API costs significantly.

Streaming Support

Process records line-by-line without buffering entire datasets.

Type Inference

Automatic detection of field types for schema generation.

Round-Trip Safe

Lossless encoding in strict mode. Compact mode for maximum savings.

Quick Syntax Reference

#

Array

Declares array with count: items#3

^

Schema

Defines fields: #3^id,name,active

@

Object

Key-value pairs: @id:1 name:Alice

T/F

Boolean

Compact true/false representation

N

Null

Compact null representation

K/M

Numbers

Abbreviated: 95K, 2.5M, 1B
